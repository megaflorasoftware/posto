## Key features

<CardGrid>
  <Card title="Editor and preview side by side" icon="laptop">
    Edit content on one side and see the running site on the other. See [Editing
    a site](/features/editing-a-site/) and [Previewing a
    site](/features/previewing-a-site/).
  </Card>
  <Card title="Local environment setup" icon="setting">
    The desktop app detects and installs Node, the package manager, and
    dependencies, then runs the site's dev server, so a non-programmer can get a
    site running the way a developer would.
  </Card>
  <Card title="Native Astro support" icon="astro">
    Rich-text `.mdx` editing with a site's own components, forms generated from
    [content
    collections](https://docs.astro.build/en/guides/content-collections/), and
    [media libraries](/frameworks/astro/media-libraries/) built from image
    collections. See [Typed collections](/frameworks/astro/typed-collections/).
  </Card>
  <Card title="Publishing" icon="github">
    Commit and push changes without running Git directly, and preview how a page
    appears in search results and when shared. See [Publishing a
    site](/features/publishing-a-site/).
  </Card>
</CardGrid>
