## Key features

<CardGrid>
  <Card title="Content and preview side by side" icon="laptop">
    Edit frontmatter and page content in one continuous workspace while the running site updates beside it. See [Editing a site](/features/editing-a-site/) and [Previewing a site](/features/previewing-a-site/).
  </Card>
  <Card title="Local environment setup" icon="setting">
    The desktop app installs the required Node version, package manager, and dependencies, then starts the site's development server.
  </Card>
  <Card title="Native Astro support" icon="astro">
    Edit `.mdx` with the site's own components, generate forms from [content collections](https://docs.astro.build/en/guides/content-collections/), and manage image collections as [media libraries](/frameworks/astro/media-libraries/).
  </Card>
  <Card title="Media management" icon="seti:image">
    Import, organize, and reuse public media or metadata-backed image libraries. Drag media into content and move it between folders without repairing references by hand. See [Managing site media](/features/managing-site-media/).
  </Card>
  <Card title="Publishing" icon="github">
    Review, commit, and push changes without running Git directly. Then follow a GitHub Actions deployment from Posto. See [Publishing a site](/features/publishing-a-site/).
  </Card>
</CardGrid>
